## 地图页高保真草图与规范

### 布局
- 桌面：左侧列表 360px + 右侧地图自适应；移动：地图全屏，底部抽屉式列表。
- 顶部：筛选条（运动类型、价格、室内外、设施、评分），右侧定位按钮。

### 地图元素
- Marker：运动类型区分样式（绿-篮球、蓝-足球），聚合泡泡显示数量。
- InfoWindow：封面小图、名称、评分、价格/免费、导航与详情按钮。
- 我的位置：蓝点；范围搜索（按视野、圆形半径）。

### 列表卡片
- 紧凑模式：小图+标题+评分+距离+价格；Hover 同步高亮地图标记。

### 交互
- 地图拖动或缩放后出现“在此搜索”浮层按钮。
- 点击列表项 → 地图定位并弹出InfoWindow。
