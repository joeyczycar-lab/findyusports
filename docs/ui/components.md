## 关键组件规范

### 场地卡片（VenueCard）
- 结构：封面图、收藏按钮、标题、评分、价格/免费、标签、距离。
- 手势：Hover 提升阴影，点击进入详情；收藏切换需要登录。

### 筛选面板（Filters）
- 选项：运动类型、价格区间、室内外、设施、评分、是否可预约。
- 模式：横向条 + 弹出层；移动端为底部抽屉。

### 地图标记（MapMarker）
- 颜色：篮球primary绿、足球brand蓝；聚合泡泡按数量深浅。
- 状态：选中高亮 + 脉冲。

### 导航按钮（NavigateButton）
- 主色填充，图标：方向箭头；点击弹出选择地图App。

### 点评单元（ReviewItem）
- 头像、昵称、评分星、时间、内容、图片墙、标签。
