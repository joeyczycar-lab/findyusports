# 解决管理页面空白问题

## 问题描述
刷新管理页面（如 `/admin/venues`、`/admin/data`）时显示空白页面。

## 可能的原因

1. **Next.js 构建缓存问题** - 最常见
2. **错误组件未正确加载**
3. **服务未完全启动**

## 解决方案

### 方法 1: 清除缓存并重启（推荐）

```bash
cd Web/webapp

# 1. 停止前端服务（在运行前端的终端按 Ctrl+C）

# 2. 清除所有缓存
rm -rf .next node_modules/.cache .swc

# 3. 重新启动
npm run dev
```

### 方法 2: 使用清除脚本

```bash
cd /Users/Zhuanz/Documents/findyusports
./彻底清除Next.js缓存.sh
```

### 方法 3: 等待服务完全启动

Next.js 开发服务器需要时间编译所有页面：
1. 等待 30-60 秒
2. 刷新浏览器页面（Cmd+Shift+R 或 Ctrl+Shift+R）
3. 检查浏览器控制台（F12）查看是否有错误

## 已创建的组件

- ✅ `src/app/error.tsx` - 页面级错误处理
- ✅ `src/app/not-found.tsx` - 404 错误处理  
- ✅ `src/app/global-error.tsx` - 全局错误处理

## 如果问题持续

1. **检查前端服务终端输出** - 查看是否有编译错误
2. **检查浏览器控制台** - 查看是否有 JavaScript 错误
3. **尝试重新安装依赖**：
   ```bash
   cd Web/webapp
   rm -rf node_modules
   npm install
   rm -rf .next
   npm run dev
   ```

## 注意事项

- "missing required error components" 错误有时是 Next.js 开发模式的临时状态
- 等待服务完全启动后再访问页面
- 如果构建成功但页面仍空白，可能是浏览器缓存问题，尝试硬刷新
