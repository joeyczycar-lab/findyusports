# 如何检查按钮圆角

## 正确的检查方法

### 步骤 1：选择按钮元素（不是 body）

1. **在开发者工具中**：
   - 点击左上角的"选择元素"图标（或按 `Cmd + Shift + C`）
   - 然后**点击页面上的任意按钮**（如"开始探索"、"篮球"、"添加场地"等）

2. **或者手动选择**：
   - 在 Elements 面板中，展开 `<body>` → `<main>` → 找到 `<a>` 或 `<button>` 元素
   - 点击按钮元素（不是 body）

### 步骤 2：查看按钮的样式

选中按钮后，在右侧样式面板中：

1. **搜索 "border-radius"**：
   - 在样式面板的搜索框中输入 `border-radius`
   - 应该能看到 `border-radius: 2px` 或 `border-radius: 0.125rem`（2px = 0.125rem）

2. **查看计算样式**：
   - 点击"计算样式"（Computed）标签
   - 搜索 `border-radius`
   - 查看实际计算值

### 步骤 3：检查是否有覆盖

如果看到 `border-radius: 0` 或其他值：
- 查看是否有其他样式规则覆盖了它（显示为删除线）
- 检查是否有内联样式覆盖

## 快速测试方法

在浏览器控制台（Console）中运行：

```javascript
// 检查所有按钮的 border-radius
document.querySelectorAll('.btn-primary, .btn-secondary, button, a[href="/map"]').forEach(btn => {
  const style = window.getComputedStyle(btn)
  console.log(btn.textContent.trim(), ':', style.borderRadius)
})
```

这会显示所有按钮的实际 `border-radius` 值。
