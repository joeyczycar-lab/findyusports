导航栏调试工具

在浏览器控制台中运行以下代码来检查导航栏状态：

1. 检查导航栏元素是否存在

const header = document.querySelector('header');
console.log('导航栏元素:', header);
console.log('是否存在:', !!header);

2. 检查导航栏样式

const header = document.querySelector('header');
if (header) {
  const styles = window.getComputedStyle(header);
  console.log('位置:', header.getBoundingClientRect());
  console.log('display:', styles.display);
  console.log('position:', styles.position);
  console.log('z-index:', styles.zIndex);
  console.log('visibility:', styles.visibility);
  console.log('opacity:', styles.opacity);
  console.log('height:', styles.height);
}

3. 强制显示导航栏（测试用）

const header = document.querySelector('header');
if (header) {
  header.style.cssText = `
    position: fixed !important;
    top: 0 !important;
    left: 0 !important;
    right: 0 !important;
    width: 100% !important;
    height: 64px !important;
    background-color: red !important;
    z-index: 99999 !important;
    display: flex !important;
    visibility: visible !important;
    opacity: 1 !important;
  `;
  console.log('导航栏已强制显示为红色背景');
}

4. 检查是否有多个导航栏

const headers = document.querySelectorAll('header');
console.log('导航栏数量:', headers.length);
headers.forEach((h, i) => {
  console.log(`导航栏 ${i + 1}:`, h.getBoundingClientRect());
});

5. 检查 body 和 html 样式

const bodyStyles = window.getComputedStyle(document.body);
const htmlStyles = window.getComputedStyle(document.documentElement);
console.log('body margin-top:', bodyStyles.marginTop);
console.log('body padding-top:', bodyStyles.paddingTop);
console.log('html scroll-padding-top:', htmlStyles.scrollPaddingTop);
