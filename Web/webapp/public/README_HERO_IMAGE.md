# 首页背景图片说明

## 图片位置
请将首页背景图片保存为：
`Web/webapp/public/hero-background.jpg`

## 图片要求
- 文件名：`hero-background.jpg`
- 位置：`Web/webapp/public/` 目录下
- 建议尺寸：1920x1080 或更大（宽屏比例）
- 格式：JPG 或 PNG

## 使用方法
1. 将您的图片文件重命名为 `hero-background.jpg`
2. 放到 `Web/webapp/public/` 目录下
3. 刷新页面即可看到效果

## 当前状态
代码已更新为使用 `/hero-background.jpg` 作为背景图片。
如果图片不存在，背景将显示为黑色。
