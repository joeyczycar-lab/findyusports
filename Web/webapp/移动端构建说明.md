# Find遇体育 - 移动端 App 构建说明

当前 App 为 **Capacitor 壳**，壳内加载线上站 **https://findyusports.com**，与网页端数据完全同步。

---

## 一、环境要求

### Android
- **JDK 17**（推荐 [Adoptium](https://adoptium.net/) 或 Oracle JDK）
- **Android SDK**：推荐直接安装 [Android Studio](https://developer.android.com/studio)，会自动安装 SDK（macOS 默认在 `~/Library/Android/sdk`）
- **指定 SDK 路径**：在 `Web/webapp/android/` 下创建 `local.properties`，内容为一行：`sdk.dir=/Users/你的用户名/Library/Android/sdk`（或参考 `local.properties.example`）

### iOS（仅 macOS）
- **Xcode**（从 App Store 安装）
- **CocoaPods**：`sudo gem install cocoapods`

---

## 二、生成 Android App（APK）

### 1. 同步 Capacitor（每次改配置或插件后建议执行）
```bash
cd Web/webapp
npm run cap:sync
```

### 2. 构建调试版 APK（可安装到手机测试）
```bash
cd Web/webapp
npm run cap:build:android
```
- 成功后 APK 位置：`Web/webapp/android/app/build/outputs/apk/debug/app-debug.apk`
- 用数据线连接手机并开启「USB 调试」，或把 APK 传到手机安装

### 3. 构建正式版 APK（用于上架或分发）
```bash
npm run cap:build:android:release
```
- 输出：`android/app/build/outputs/apk/release/app-release.apk`
- 上架应用商店前需在 Android Studio 里配置签名（Build > Generate Signed Bundle / APK）

### 4. 用 Android Studio 打开项目（可选）
```bash
npm run cap:android
```
- 在 Android Studio 中：Build > Build Bundle(s) / APK(s) > Build APK(s)
- 可在此配置签名、版本号、混淆等

---

## 三、生成 iOS App（仅 macOS）

### 1. 同步并安装依赖
```bash
cd Web/webapp
npm run cap:sync
cd ios/App && pod install && cd ../..
```

### 2. 用 Xcode 打开并构建
```bash
npm run cap:ios
```
- 在 Xcode 中选择真机或模拟器，点击 Run（▶️）即可运行
- 归档发布：Product > Archive，再通过 Organizer 上传 App Store 或导出 IPA

---

## 四、修改 App 行为

- **壳加载的地址**：编辑 `Web/webapp/capacitor.config.ts` 中的 `server.url`，例如改为 `http://localhost:3000` 可加载本地前端（仅开发/调试用）。
- **应用名、包名**：`capacitor.config.ts` 中 `appName`、`appId`；Android 包名在 `android/app/build.gradle` 的 `applicationId`，iOS 在 Xcode 的 target 与 Info.plist 中修改。

---

## 五、常见问题

| 问题 | 处理 |
|------|------|
| `Unable to locate a Java Runtime` | 安装 JDK 17 并设置 `JAVA_HOME` 或将 OpenJDK 的 bin 加入 PATH |
| **SDK location not found** | 1）安装 [Android Studio](https://developer.android.com/studio)；2）在 `android/` 下创建 `local.properties`，写一行 `sdk.dir=/Users/你的用户名/Library/Android/sdk`（可复制 `local.properties.example` 为 `local.properties` 再改路径） |
| Android 构建报错 SDK 未找到 | 用 Android Studio 安装 Android SDK（含 Build-Tools、对应 compileSdkVersion） |
| iOS 构建失败 / 签名错误 | 在 Xcode 中为项目选择你的 Apple 开发团队与证书 |
| 白屏或无法打开网页 | 检查设备网络；确认 `capacitor.config.ts` 里 `server.url` 可访问（如 https://findyusports.com） |

完成上述任一步骤后，即可在真机或模拟器上运行/安装当前移动端 App。
