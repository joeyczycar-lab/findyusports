# 添加场地 - Web 管理界面

## 访问方式

1. **通过导航栏**：点击网站顶部导航栏的"添加场地"链接
2. **直接访问**：`https://findyusports.com/admin/add-venue`

## 使用步骤

### 1. 填写基本信息

- **场地名称**（必填）：例如"朝阳体育中心篮球场"
- **运动类型**（必填）：选择"篮球"或"足球"
- **城市**（必填）：从下拉列表选择城市

### 2. 填写地址和坐标

- **详细地址**（可选）：例如"北京市朝阳区朝阳路1号"
- **经度 (lng)**（必填）：例如 `116.45`
- **纬度 (lat)**（必填）：例如 `39.92`

**如何获取坐标？**

点击"获取坐标 →"链接，会打开高德地图坐标拾取工具：
1. 在地图上找到你的场地位置
2. 点击地图上的位置
3. 复制显示的经纬度
4. 粘贴到表单中

> ⚠️ **注意**：高德地图使用 GCJ-02 坐标系，如果坐标偏差较大，可能需要转换为 WGS-84 坐标系。

### 3. 填写价格信息（可选）

- **最低价格**：元/小时
- **最高价格**：元/小时

### 4. 选择场地类型

- 勾选"室内场地"如果场地在室内

### 5. 提交

点击"添加场地"按钮，系统会：
- ✅ 验证所有必填字段
- ✅ 提交数据到后端 API
- ✅ 显示成功或错误消息
- ✅ 成功后自动清空表单，可继续添加

## 常见问题

### Q: 添加失败怎么办？

1. **检查网络连接**：确保能访问后端 API
2. **检查后端服务**：确保后端服务正在运行
3. **检查输入格式**：确保经纬度是有效数字
4. **查看错误消息**：页面会显示具体的错误信息

### Q: 坐标不准确怎么办？

- 使用高德地图坐标拾取工具获取更准确的坐标
- 如果使用其他地图服务，注意坐标系转换（GCJ-02 → WGS-84）

### Q: 可以批量添加吗？

目前 Web 界面只支持单个添加。如需批量添加，请使用：
- 交互式命令行工具：`npm run add-venue:interactive`
- JSON 文件批量导入：`npm run add-venue -- --file venues.json`

## 技术说明

- **API 端点**：`POST /venues`
- **认证**：无需认证（公开访问）
- **数据验证**：前端和后端双重验证
- **错误处理**：友好的错误提示
